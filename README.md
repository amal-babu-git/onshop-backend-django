# OnShop Backend

OnShop is a e-commerce web application
build with Django REST backend and react frontend.

> This is backend repo

## Main Frontend Packages 
* ReactJS
* ReduxJS
* MDBootstrap

## Backend
* Django
* Django REST
* jwt auth
* PostgreSQL
* Online payment - Braintree & Razorpay
* Custom filters
* Docker
* https(SSL) with Letsencrypt-certbot
